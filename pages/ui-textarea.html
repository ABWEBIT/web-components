Textarea

Textarea allows users to enter and edit multiline text.

Usage

  • Static init-only attributes are removed from the custom element itself immediately after initialization.
    To update these values dynamically, use the corresponding properties on the component instance.

Features

Attributes

  • placeholder - static (optional).
    Text displayed when the textarea is empty. Update via property: component.placeholder = 'Placeholder'.
  • required - static (optional).
    Defines whether the component is required.
  • disabled - reactive (optional).
    Controls the disabled state of the component.
  • shape - static (optional). Default: rounded.
    Available options: rounded, square.
    Sets the corner style.
  • size - static (optional). Default: default.
    Specifies the component's size (extendable via CSS).
  • color - static (optional). Default: default.
    Specifies the component's color scheme (extendable via CSS).
  • width-mode="full" - static (optional).
    Expands the component to take up 100% of the available width.
