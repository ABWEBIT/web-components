Textarea

Textarea allows users to enter and edit multiline text.

Usage

  • Static init-only attributes are removed from the custom element itself immediately after initialization. This is done to prevent confusion and avoid conflicts between attributes and properties during the component lifecycle. Their values are still applied to the relevant internal elements, so the component's behavior remains consistent with the initial configuration.

    To update these values dynamically, use the corresponding JavaScript properties on the component instance.

Features

Attributes

  • placeholder - static (init-only), optional.
    Text displayed when the textarea is empty. Update via property: component.placeholder = 'Placeholder'.
  • height - static, optional. Default: 40.
    Component height is measured in pixels. Recommended range: 24 to 40. Even values are preferred.
  • clearable - static, optional.
    Adds a clear button, which, when clicked, removes all content from the form control.
  • required - static, optional.
    Defines whether the component is required.
  • disabled - reactive, optional.
    Controls the disabled state of the component.
  • shape - static, optional. Default: rounded.
    The component shape can be rounded or square.
