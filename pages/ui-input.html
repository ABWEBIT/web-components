Поле ввода
Поле ввода позволяет пользователю вводить и редактировать текст.
  • Поддерживает LTR и RTL.
  • type - тип поля, по умолчанию text
  • placeholder - статичный, необязательный атрибут. Текст заглушки.
  • icon-leading и icon-trailing — реактивные, необязательные атрибуты. Определяют имена иконок, отображаемых соответственно до и после поля.
  • height - статичный, необязательный атрибут. Высота поля (число), по умолчанию - 32, измеряется в пикселях. (рекомендуется от 24 до 40). Лучше использовать чётные числа.
  • Размер шрифта изменяется автоматически, рассчитывается по формуле "выcота кнопки / 2"
  • disabled - реактивный, необязательный атрибут, управляющий состоянием компонента (отключён/включён).
  • required — статичный, необязательный атрибут, управляющий состоянием компонента (обязательное или необязательное заполнение).
