Input Field
The input field allows the user to enter and edit text.
  • Supports LTR and RTL.
  • type - the type of input, default is text.
  • placeholder - static, optional attribute. Placeholder text.
  • icon-leading and icon-trailing - reactive, optional attributes. Define the icon names displayed before and after the field, respectively.
  • height - static, optional attribute. Field height (number), default is 32, measured in pixels. (Recommended: 24 to 40). Even numbers are preferred.
  • Font size adjusts automatically, calculated by the formula "button height / 2".
  • clearable - static, optional attribute that adds a clear button to the field.
  • disabled - reactive, optional attribute that controls the componentâ€™s state (enabled/disabled).
  • required - static, optional attribute that defines whether the field is required or not.
  • Font size is calculated as "button height / 2 - 2", but no less than 12px and no more than 16px.
  • Corner rounding is calculated as "button height / 4".
