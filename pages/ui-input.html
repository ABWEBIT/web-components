Input

Allows users to enter and edit text.

Features

  • Supports LTR and RTL.
  • Font size is "button height / 2", capped between 12px and 16px.
  • Corner rounding is calculated as "button height / 4".
  • A visible outline appears on focused input.

Attributes

  • type - static, optional. Default: "text".
  • placeholder - static, optional.
  • icon-leading and icon-trailing - reactive, optional.
    Define the icons displayed before and after the label, respectively.
  • height - static, optional. Default: 40.
    Component height is measured in pixels. Recommended range: 24 to 40. Even values are preferred.
  • clearable - static, optional.
    Adds a clear button.
  • required - static, optional.
    Defines whether the input is required or not.
  • disabled - reactive, optional.
    Controls the disabled state of the component.
  • shape - static, optional. Default: rounded.
    A button can be rounded, pill, or square.
