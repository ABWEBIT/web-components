Input

Allows users to enter and edit text.

Features

  • Supports LTR and RTL.
  • Font size is "button height / 2", capped between 12px and 16px.
  • Corner rounding is calculated as "button height / 4".
  • A visible outline appears on focused input.

Attributes

  • type - static, optional. Default: "text".
  • placeholder - static, optional.
  • icon-leading and icon-trailing - reactive, optional. Define the icon names.
  • height - static, optional. Default: 32, measured in pixels. Recommended: 24 to 40 pixels. Even numbers are preferred.
  • clearable - static, optional. Adds a clear button.
  • disabled - reactive, optional. Controls the component's state (enabled/disabled).
  • required - static, optional. Defines whether the input is required or not.
