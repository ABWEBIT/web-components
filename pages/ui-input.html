Input

Input allows users to enter and edit text.

Features

  • The clickable area of the Clear button is larger than the icon itself to make it easier to click or tap.

Attributes

  • type - static, optional. Default: text.
    Specifies the input type. This attribute is removed after initialization; subsequent changes should be done via the component's type property (e.g., component.type = "email").
  • value - static, optional.
    Sets the current value of the input. This attribute is removed after initialization; further interaction is handled through the component's value property (e.g., component.value = "new value").
  • placeholder - static, optional.
    Text displayed when the input is empty. This attribute is removed after initialization; use the placeholder property to update it dynamically (e.g., component.placeholder = "enter text here").
  • clearable - static, optional.
    Adds a clear button to the input.
  • required - static, optional.
    Indicates whether the input is required.
  • disabled - reactive, optional.
    Controls the disabled state of the component and reflects changes both as an attribute and property.
  • size - static, optional. Default: medium.
    Available values: small, medium, large.
  • shape - static, optional. Default: rounded.
    Defines the shape of the input corners: rounded, pill, or square.
