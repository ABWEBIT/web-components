Foundation

Key fundamental capabilities of the UI library.

Information

  • The library does not use Shadow DOM.
  • The library does not use internal CSS class names.
  • Text direction follows the parent or system context unless explicitly set.

Features

  • Supports both LTR (Left-to-Right) and RTL (Right-to-Left) text directions.

Attributes

  • ARIA attributes used are valid for their roles and accurately reflect the component's state.
