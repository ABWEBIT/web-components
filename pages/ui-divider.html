Разделитель Компонент для разделения контента с помощью линии или линии с текстом.
  • Поддерживает LTR и RTL.
  • label - необязательный атрибут, непрерывная строка.
  • axis - "x" или "y" (строчные), определяет ось. По умолчанию - x.
  • axis="x" - горизонтальный разделитель, ширина 100%.
  • axis="y" - вертикальный разделитель, высота 100%.
  • text-align - выравнивание текста: "start" и "end". По умолчанию - по центру.
