Разделитель Компонент для разделения контента с помощью линии или линии с текстом.
  • label - необязательный атрибут (тип строка).
  • axis - "x" или "y" (строчные), отвечает за направление. По умолчанию x.
  • axis="x" - горизонтальный разделитель, ширина 100%.
  • axis="y" - вертикальный разделитель, высота 100%.
  • text-align - расположение текста. По умолчанию по центру.
  • text-align - "left" или "right" работают только с axis="x".
  • text-align - "top" или "bottom" работают только с axis="y".
