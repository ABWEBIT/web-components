Divider

A component for separating content using a line or a line with text.

Features

  • Supports LTR and RTL.

Attributes

  • label - optional attribute, a continuous string.
  • axis - defines the axis. Default is "x" (lowercase).
  • axis="x" - horizontal divider, 100% width.
  • axis="y" - vertical divider, 100% height.
  • text-align - text alignment: "start" or "end". Center by default.
