Divider

A component for visually separating content with a line that may contain additional elements such as text or icons.

Usage

  • Any content added inside the component should be wrapped in a <span> or <div> to ensure proper rendering.
  • When using non-text content (e.g., icons or custom elements), additional CSS may be required to ensure correct styling.

Features

  • By default, a horizontal divider has a width of 100%.
  • By default, a vertical divider has a height of 100%.

Attributes

  • content-align - static (optional). Default: center.
    Available options: start, center, end.
  • orientation - static (optional). Default: horizontal.
    Available options: horizontal, vertical.
    Defines the divider direction.
