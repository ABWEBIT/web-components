Divider

A component for separating content using a line or a line with text.

Features

  • By default, horizontal divider width is 100%.
  • By default, vertical divider height is 100%.

Attributes

  • text - static, optional.
    Continuous string. (nowrap)
  • text-align - static, optional. Default: center.
    Text alignment: start, end.
  • orientation - static, optional. Default: horizontal.
    Defines the divider direction. horizontal | vertical
