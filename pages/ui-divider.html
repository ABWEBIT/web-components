Разделитель Компонент для разделения контента с помощью линии или линии с текстом.
  • Все атрибуты статичные.
  • Горизонтальный разделитель имеет ширину 100%.
  • Вертикальный разделитель имеет высоту 100%.
  • label - не обазательный атрибут (тип строка), никаких ограничений.
  • axis - x или y (строчные), отвечает за направление. По умолчанию x.
  • align - расположение текста. По умолчанию по центру.
  • align - left или right работают только с axis="x".
  • align - top или bottom работают только с axis="y".
