Divider

A component for separating content using a line or a line with text.

Features

  • Supports LTR and RTL.

Attributes

  • label - reactive, optional. Continuous string. (nowrap)
  • axis - static. Divider direction (x | y). Default: x (width: 100%); y (height: 100%).
  • text-align - text alignment: "start"; "end". Default: center.
