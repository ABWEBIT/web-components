Alert

Displays a message that draws the user's attention.

Info

Neither element is mandatory; however, either a title or a content is required to display a meaningful message.

This component is designed to work with the icon component for rendering icons.

Object properties

label - optional. Alert label displayed at the top. Plain text only.

content - optional. Additional alert content, which can include plain text or HTML markup.

Attributes

non-closable - static (optional, boolean). Hides the close button when the attribute is present.

icon - static (optional). Adds an icon by specifying its name in the attribute, e.g., icon="name".

Alert with content, without a close button
HTML 

<ui-alert id="alert" non-closable></ui-alert>

alert.data = {
 content:'Your settings have been <strong>successfully</strong> saved.'
};
Alert with label and content
HTML 

<ui-alert id="alert"></ui-alert>

alert.data = {
 label:'Update Complete',
 content:'Your settings have been <strong>successfully</strong> saved.'
};
Alert with label, content, close button, and icon
HTML 

<ui-alert id="alert" icon="info"></ui-alert>

alert.data = {
 label:'Update Complete',
 content:'Your settings have been <strong>successfully</strong> saved.'
};
