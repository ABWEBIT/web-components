Accordion

Accordion toggles collapsible content sections.

Attributes

shape - static (optional). Default: rounded. Available options: rounded, square. Sets the corner style.

Object properties

label - required. The text displayed in the accordion header.

content - required. Additional panel content, which can include plain text or HTML markup.

disabled - optional. If true, the item is non-interactive and cannot be toggled.

expanded - optional. If true, the panel is expanded by default on initial render.
