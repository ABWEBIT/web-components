Switch

Switches provide a simple and user-friendly way to toggle between two states.

Features

  • Supports LTR and RTL.

Attributes

  • label - reactive, optional.
    Continuous string. (nowrap)
  • aria-label - static, required.
    This attribute is added automatically for buttons with a label attribute and you can override it manually. If there is no label, the aria-label must be set manually.
  • height - static, optional. Default: 32.
    Switch height (number). Measured in pixels. Recommended: 16 to 24. Even numbers are preferable.
  • disabled - reactive, optional.
    Controls whether the component is enabled or disabled.
