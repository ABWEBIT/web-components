Switch

Switches provide a simple and user-friendly way to toggle between two states.

Features

  • A visible outline appears on focused buttons.
  • Supports keyboard navigation: Tab to move, Enter/Space to trigger action.

Attributes

  • aria-label - static required.
    If used outside of the field component, the aria-label must be set manually.
  • checked - reactive, required. Default: false.
    Controls the checked state of the component.
  • required - static, optional.
    Indicates whether the component is required.
  • disabled - reactive, optional.
    Controls the disabled state of the component.
