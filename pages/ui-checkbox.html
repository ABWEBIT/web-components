Checkbox

Checkboxes allow users to select multiple items from a list or toggle a single option on or off.

Features

  • Supports LTR and RTL.
  • The following ARIA attributes are added automatically: aria-disabled and aria-checked.
  • A visible outline appears on focused buttons.
  • Supports keyboard navigation: Tab to move, Enter/Space to trigger action.

Attributes

  • aria-label - static, required.
    The aria-label must be set manually.
  • height - static, optional. Default: 20.
    Component height is measured in pixels. Recommended range: 16 to 24. Even values are preferred.
  • checked - reactive, optional.
    Controls the checked state of the component.
  • disabled - reactive, optional.
    Controls the disabled state of the component.
