Checkbox

Checkboxes allow users to select multiple items from a list or toggle a single option on or off.

Features

  • A visible outline appears when the component is focused.
  • Supports keyboard navigation: Tab to move, Enter/Space to trigger action.

Attributes

  • aria-label - static (required).
    If used outside of the field component, the aria-label must be set manually.
  • checked - reactive (optional).
    Controls the checked state of the component.
  • required - static (optional).
    Indicates whether the component is required.
  • disabled - reactive (optional).
    Controls the disabled state of the component.
  • shape - static (optional). Default: rounded.
    Sets the corner style.
    Available options: rounded, circle, square.
  • size - static (optional). Default: small.
    Available options: x-small, small, medium.
