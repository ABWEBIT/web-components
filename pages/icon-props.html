Параметры иконки
Атрибут Тип Обязательный Информация
name Реактивный Да Латиница и цифры, начало с буквы. a-z A-Z 0-9
width Реактивный Нет Целое или дробное число и px | %
height Реактивный Нет Целое или дробное число и px | %
color Реактивный Нет CSS переменная --theme-dark
Примеры
<icon-block name="icon"></icon-block> <icon-block name="iconName24" width="24px" height="50%" color="--theme-dark"></icon-block>
