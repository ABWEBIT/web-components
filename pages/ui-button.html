Кнопка Компонент, который может выступать в роли ссылки или запускать действие.
  • Требуется хотя бы один из атрибутов: label, icon-left, icon-right.
  • Для создания кнопки с одной иконкой, можно использовать icon-left или icon-right.
  • label - реактивный, не обязательный атрибут. Текст кнопки, непрерывная строка.
  • icon-left - реактивный, не обязательный атрибут. Имя иконки расположенной по левому краю.
  • icon-right - реактивный, не обязательный атрибут. Имя иконки расположенной по правому краю.
  • height - статичный, не обязательный атрибут. Высота кнопки (число), по умолчанию - 32, измеряется в пикселях. (рекомендуется от 24 до 40). Лучше использовать чётные числа.
  • Размер шрифта изменяется автоматически, рассчитывается по формуле "выcота кнопки / 2"
  • text-align - "left"или "right" расположение текста. По умолчанию по центру.
  • disabled - реактивный, не обязательный атрибут, отвечает за состояние.
