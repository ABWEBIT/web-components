Button

Buttons are used to perform actions or submit forms.

Usage

  • The component acts as both a button and a wrapper. It can contain text, icons, or both - in any order you prefer.
  • To make the button square, use aspect-ratio="1/1".
  • More customization options can be added via the CSS.

Attributes

  • aria-label - static (required).
    Must be set manually.
  • disabled - reactive (optional).
    Controls the disabled state of the component.
  • loading - reactive (optional).
    Controls the loading state of the component.
  • size - static (optional). Default: default.
    Specifies the component's size (extendable via CSS).
  • color - static (optional). Default: default.
    Specifies the component's color scheme (extendable via CSS).
  • shape - static (optional). Default: rounded.
    Available options: rounded, pill, square, circle.
    Sets the corner style.
  • aspect-ratio="1/1" - static (optional).
    Makes the component square using a 1:1 aspect ratio.
  • width-mode="full" - static (optional).
    Expands the component to take up 100% of the available width.
