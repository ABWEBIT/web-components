Button

Buttons are used to perform actions or submit forms.

Usage

  • To create a button, you must include at least one of the following inside <ui-button>:
    • plain text (e.g., Submit)
    • a <ui-text> component
    • a <ui-icon> component
    You can combine them freely - for example, both text and icon together.

    If you're using multiple elements (e.g. an icon and text), it's recommended to wrap the text inside a <ui-text> component for consistent styling, spacing, and alignment.

Attributes

  • aria-label - static (required).
    Must be set manually for accessibility.
  • disabled - reactive (optional).
    Controls the disabled state of the component.
  • loading - reactive (optional).
    Controls the loading state of the component.
  • size - static (optional). Default: medium.
    Available values: small, medium, large.
  • shape - static (optional). Default: rounded.
    Sets the corner style: rounded, pill, sharp, or circle.
  • color - static (optional). Default: gray.
    Specifies the color scheme.
  • display - static (optional).
    Defines component dimensions.

    Available options:
    • width-full - sets width to 100%
    • aspect-square - enforces 1:1 aspect ratio
