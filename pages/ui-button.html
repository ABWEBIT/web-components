Кнопка Компонент, который может выступать в роли ссылки или запускать действие.
  • Поддерживает LTR и RTL.
  • Требуется хотя бы один из атрибутов: label, icon-start, icon-end.
  • label - реактивный, необязательный атрибут. Текст кнопки, непрерывная строка.
  • icon-start и icon-end — реактивные, необязательные атрибуты. Определяют имена иконок, отображаемых соответственно до и после текста.
  • Для создания кнопки с одной иконкой, можно использовать icon-start или icon-end.
  • height - статичный, необязательный атрибут. Высота кнопки (число), по умолчанию - 32, измеряется в пикселях. (рекомендуется от 24 до 40). Лучше использовать чётные числа.
  • Размер шрифта изменяется автоматически, рассчитывается по формуле "выcота кнопки / 2"
  • disabled - реактивный, необязательный атрибут, управляющий состоянием компонента (отключён/включён).
Высота 24.
<ui-button icon-start="gear" height="24"></ui-button> <ui-button label="Кнопка" height="24"></ui-button>

Неактивная кнопка.

Текст и иконка с одной стороны.
<ui-button icon-start="gear" label="Кнопка"></ui-button> <ui-button icon-end="gear" label="Кнопка"></ui-button>
Текст с иконками по обе стороны.
<ui-button icon-start="gear" icon-end="send" label="Кнопка"></ui-button>
