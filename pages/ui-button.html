Button

A component that triggers an action.

Features

  • Supports LTR and RTL.
  • The following ARIA attributes are added automatically: aria-disabled, aria-busy.
  • At least one of these attributes is required to create a button: label, icon-leading, or icon-trailing.
  • For a button with a single icon, use either icon-leading or icon-trailing.
  • Font size is "button height / 2 - 2px", capped between 12px and 16px.
  • Corner rounding is calculated as "button height / 4".
  • A visible outline appears on focused buttons.
  • Supports keyboard navigation: Tab to move, Enter/Space to trigger action.

Attributes

  • label - reactive, optional.
    Continuous string. (nowrap)
  • aria-label - dynamic, required.
    The aria-label is auto-set from label (if present), but can be overridden. If there's no label, it must be set manually.
  • icon-leading and icon-trailing - reactive, optional.
    Define the icons displayed before and after the label, respectively.
  • height - static, optional. Default: 24.
    The component's height is measured in pixels. Recommended range: 24 to 40. Even values are preferred.
  • disabled - reactive, optional.
    Controls the disabled state of the component.
  • loading - reactive, optional.
    Controls the loading state of the component.
  • shape - static, optional. Default: rounded.
    A button can be rounded, pill, or square.
