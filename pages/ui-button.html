Button A component that triggers an action.
  • Supports LTR and RTL.
  • To create a button, at least one of the attributes is required: label, icon-leading, icon-trailing.
  • To create a button with a single icon, you can use icon-leading or icon-trailing.
  • label - reactive, optional attribute. Button text, a continuous string.
  • icon-leading and icon-trailing - reactive, optional attributes. Define the icon names displayed before and after the text.
  • height - static, optional attribute. Button height (number), default is 32, measured in pixels. (Recommended: 24 to 40). Even numbers are preferable.
  • disabled - reactive, optional attribute that controls the component's state (enabled/disabled).
  • Font size is calculated as "button height / 2", but no less than 12px and no more than 18px.
  • Corner rounding is calculated as "button height / 4".
