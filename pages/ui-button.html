Кнопка Компонент, который может выступать в роли ссылки или запускать действие.
  • Требуется хотя бы один из атрибутов: label, icon-left, icon-right.
  • Для создания кнопки с одной иконкой, можно использовать icon-left или icon-right.
  • label - реактивный, не обязательный. Текст кнопки.
  • icon-left - реактивный, не обязательный. Имя иконки расположенной по левому краю.
  • icon-right - реактивный, не обязательный. Имя иконки расположенной по правому краю.
  • height - статичный, не обязательный. Высота иконки, по умолчанию 32px.
  • disabled - реактивный, не обязательный. Состояние кнопки - неактивная.
Высота 24px.
 
<ui-button label="Кнопка" height="24"></ui-button> <ui-button icon-left="article" height="24"></ui-button>
