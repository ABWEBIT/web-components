Breadcrumb

Breadcrumbs show the current page's location within a navigational hierarchy.

Attributes

aria-label - static (required). Must be set manually.

Default

  1. Home
  2. Settings
  3. Notifications
  4. Article
 

<ui-breadcrumb>
  <ol>
    <li><a href="#">Home</a></li>
    <li><a href="#">Settings</a></li>
    <li><a href="#">Notifications</a></li>
    <li>Article</li>
  </ol>
</ui-breadcrumb>
