Tabs

Tabs organize content into separate panels that users can switch between.

Info

IDs for tabs and panels are generated automatically, but can be manually overridden.

The connection between a tab and its panel is handled automatically via aria-controls and aria-labelledby and should not be modified.

Attributes

Tabs:

vertical - static (optional). Sets the orientation of the tabs in the tablist to vertical.

Tab:

disabled - static (optional). Controls whether the tab is disabled.
