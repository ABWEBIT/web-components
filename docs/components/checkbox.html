Checkbox

Checkboxes are used to toggle an option on or off.

Dependencies

  • ui-icon

Attributes

aria-label - static (required). If used outside of the ui-field component, the aria-label must be set manually.

checked - reactive (optional). Default: false.

indeterminate - reactive (optional). Default: false.

required - static (optional). Default: false.

disabled - reactive (optional). Default: false. Controls the disabled state of the component.
