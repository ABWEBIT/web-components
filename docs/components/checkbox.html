Checkbox

Checkboxes are used to toggle an option on or off.

Dependencies

This component depends on the following components:

  • icon

Attributes

aria-label - static (required). If used outside of the field component, the aria-label must be set manually.

required - static (optional). Default: false. Indicates whether the component is required.

checked - reactive (optional). Default: false. Controls the checked state of the component.

disabled - reactive (optional). Default: false. Controls the disabled state of the component.
