Checkbox

Checkboxes are used to toggle an option on or off.

Attributes

options - static (optional). Additional <input> attributes (id, name, value, class).
options='{"id":"unique-id","class":"utility-class","name":"product-id","value":"on"}'

checked - reactive (optional). Default: false.

indeterminate - reactive (optional). Default: false.

required - reactive (optional). Default: false.

disabled - reactive (optional). Default: false.
