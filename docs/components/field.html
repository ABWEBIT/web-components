Field

The Field component groups a div with a form control.

Info

Allows only one label and one form control inside.

Automatic association between the label and the form control by adding a unique UUID to the label's id attribute and assigning this UUID to the form control's aria-labelledby attribute. You can override the unique UUID by adding a custom id.

Attributes

vertical - static (optional). Sets the orientation to vertical.

required - static (optional). Adds the required attribute to all child elements.
