Field

The Field component groups a div with a form control.

Info

Allows only one div and one form control inside.

Automatic association between the div and the form control by adding a unique UUID to the div's id attribute and assigning this UUID to the form control's aria-labelledby attribute. You can override the unique UUID by adding a custom id attribute to the div component.

Attributes

vertical - static (optional). Sets the orientation to vertical.

required - static (optional). Adds the required attribute to all child elements.

Default
HTML 

<ui-field>
  <label>Label</label>
  <ui-checkbox></ui-checkbox>
</ui-field>
div-position="end"
HTML 

<ui-field div-position="end">
  <label>Label</label>
  <ui-switch></ui-switch>
</ui-field>
Passive, non-clickable div
HTML 

<ui-field>
  <label>Label</label>
  <ui-switch></ui-switch>
</ui-field>
Required, vertical
HTML 

<ui-field required vertical>
  <div>Label</div>
  <ui-textarea></ui-textarea>
</ui-field>
