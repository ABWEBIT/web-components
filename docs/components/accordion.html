Accordion

Accordion toggles collapsible content sections.

Dependencies

This component depends on the following components:

  • icon

Info

The structure must be strictly followed.

The accordion allows multiple items to be opened.

The heading can be any HTML heading element from h2 to h6; h3 is recommended.

Attributes

Item:

data-expanded="true" - static (optional). The associated panel is expanded.

Button:

disabled - static (optional). Determines whether the heading button is disabled.
