Switch

Switches provide a simple and user-friendly way to toggle between two states.

Attributes

aria-label - static (required). Must be set manually.

checked - reactive (required). Default: false. Controls the checked state of the component.

disabled - reactive (optional). Controls the disabled state of the component.

required - static (optional). Indicates whether the component is required.

Default
HTML 

<ui-switch></ui-switch>
Checked
HTML 

<ui-switch checked></ui-switch>
Disabled
HTML 

<ui-switch disabled></ui-switch>
