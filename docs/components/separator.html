Separator

A component for visually separating content with a line.

Info

A vertical separator needs to be placed inside a container with display: flex or grid.

The separator stretches along its axis according to the size of its parent element.

Once the CSS Gaps Decorations Module Level 1 is fully implemented, this component can be replaced with a standard CSS property.

Attributes

orientation - static (optional). Sets the separator orientation. Available options: horizontal, vertical. Default: horizontal

CSS Properties

--separator-thickness:1px;

--separator-spacing:0;

--separator-color:var(--ui-surface-color-secondary);

Horizontal
HTML 

<ui-separator></ui-separator>
Vertical (parent container: display: flex | grid)
HTML 

<ui-separator orientation="vertical"></ui-separator>
