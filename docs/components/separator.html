Separator

A component for visually separating content with a line.

Info

The separator stretches along its axis according to the size of its parent element.

This component will become obsolete once CSS Gaps Decorations Module Level 1 is fully implemented across all major browsers.

Separators default size along their orientation axis.

Attributes

orientation - static. Sets the separator orientation. Available options: horizontal, vertical. Default: horizontal

Horizontal
HTML 

<ui-separator></ui-separator>
Vertical
HTML 

<ui-separator vertical></ui-separator>
