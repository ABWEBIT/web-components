Textarea

Textarea allows users to enter and edit multiline text.

Attributes

required - static (optional). Defines whether the component is required.

disabled - reactive (optional). Controls the disabled state of the component.

shape - static (optional). Default: rounded. Available options: rounded, square. Sets the corner style.
