Textarea

Textarea allows users to enter and edit multiline text.

Info

Static init-only attributes are removed from the custom element itself immediately after initialization. To update these values dynamically, use the corresponding properties on the component instance.

Attributes

placeholder - static (optional). Text displayed when the textarea is empty. Update via property: component.placeholder = 'Placeholder'.

required - static (optional). Defines whether the component is required.

disabled - reactive (optional). Controls the disabled state of the component.

shape - static (optional). Default: rounded. Available options: rounded, square. Sets the corner style.
