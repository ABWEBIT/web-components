Input

Input allows users to enter and edit text.

Dependencies

  • ui-icon

Attributes

aria-label - static (required). If used outside of the ui-field component, the aria-label must be set manually.

clearable - static (optional). Adds a clear button to the component.

shape - static (optional). Default: rounded. Available options: rounded, pill, square. Sets the corner style.

Default

<ui-input>
  <input />
</ui-input>
Shape (default: rounded)

<ui-input shape="rounded">
  <input />
</ui-input>

<ui-input shape="pill">
  <input />
</ui-input>

<ui-input shape="square">
  <input />
</ui-input>

<ui-input clearable>
  <input />
</ui-input>
