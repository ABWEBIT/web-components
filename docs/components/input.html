Input

Input allows users to enter and edit text.

Dependencies

  • ui-icon

Attributes

config - static (optional). Additional attributes: id, class, type, name, value, placeholder.

clearable - static (optional). Adds a clear button to the component.

required - static (optional). Default: false.

disabled - reactive (optional). Default: false.

Default


<ui-input></ui-input>

Config

Each property inside the config JSON string must be enclosed in double quotes. This applies both to the property names and their values.

<ui-input
  config='{
    "id":"unique-id",
    "class":"utility-class",
    "type":"text",
    "name":"product-id",
    "value":"value",
    "placeholder":"Enter value"
  }'>
</ui-input>

Sizes


<ui-input size="small" config='{"placeholder":"Enter value"}'></ui-input>
<ui-input size="medium" config='{"placeholder":"Enter value"}'></ui-input>
<ui-input size="large" config='{"placeholder":"Enter value"}'></ui-input>

<ui-input clearable config='{"placeholder":"Enter value"}'></ui-input>

<ui-input config='{"placeholder":"Enter value"}'></ui-input>
