Input

Input allows users to enter and edit text.

Dependencies

  • ui-icon

Attributes

config - static (optional). Additional attributes: id, class, type, name, value, placeholder.

clearable - static (optional). Adds a clear button to the component.

required - static (optional). Default: false.

disabled - reactive (optional). Default: false.

Default


<ui-input></ui-input>

Config

Each property in the JSON config must be enclosed in double quotes, both property names and their values, except for the boolean values true and false.

<ui-input
  config='{
    id:"unique-id",
    class:"utility-class",
    type:"text",
    name:"product-id",
    value:"value",
    placeholder:"Enter text"
  }'>
</ui-input>

Sizes


<ui-input size="small" config='{"placeholder":"Enter text"}'></ui-input>
<ui-input size="medium" config='{"placeholder":"Enter text"}'></ui-input>
<ui-input size="large" config='{"placeholder":"Enter text"}'></ui-input>

Clear button


<ui-input clearable config='{"placeholder":"Enter text"}'></ui-input>

Required


<ui-input required config='{"placeholder":"Enter text"}'></ui-input>

Disabled


<ui-input disabled config='{"placeholder":"Enter text"}'></ui-input>
