Input

Input allows users to enter and edit text.

Dependencies

  • ui-icon

Attributes

aria-label - static (required). Must be set manually.

name - reactive (required). Must be set manually.

value - reactive (optional). Must be set manually.

clearable - static (optional). Adds a clear button to the component.

required - static (optional). Default: false.

disabled - reactive (optional). Default: false.
