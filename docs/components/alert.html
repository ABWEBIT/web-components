Alert

Displays a message that draws the user's attention.

Info

The structure must be strictly followed.

A heading or text is required to display a meaningful message.

The heading can be any HTML heading element from h2 to h6; h3 is recommended.

Dependencies

This component depends on the icon component, which is a required part of its functionality.

Attributes

non-closable - static (optional, boolean). Hides the close button when the attribute is present.

Alert with description (no close button)
Your settings have been successfully saved.
HTML 

<ui-alert non-closable>
  <div>Your settings have been successfully saved.</div>
</ui-alert>
Alert with label and description

Your settings have been successfully saved.
HTML 

<ui-alert>
  <h3>Update Complete</h3>
