Alert

Displays a message that draws the user's attention.

Dependencies

  • ui-icon

Info

The structure must be strictly followed.

A heading or text is required to display a meaningful message.

The heading can be any element from h2 to h6; h3 is recommended.

Attributes

non-closable - static (optional). Hides the close button.

Alert with description (no close button)
Your settings have been successfully saved.
 

<ui-alert>
  <div>Your settings have been successfully saved.</div>
</ui-alert>
