Alert

Displays a message that draws the user's attention.

Info

A label or text is required to display a meaningful message.

Dependencies

This component depends on the label, text, and icon components, which are required parts of its functionality.

Attributes

non-closable - static (optional, boolean). Hides the close button when the attribute is present.

Alert with description (no close button)
Your settings have been successfully saved.
HTML 

<ui-alert non-closable>
  <div>Your settings have been successfully saved.</div>
</ui-alert>
