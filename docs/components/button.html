Button

Buttons are used to perform actions or submit forms.

Dependencies

This component depends on the following components:

  • icon

Info

The component can contain text, icons, or both - in any order.

This component dispatches a custom event "button-action" whenever it is activated via click or keyboard (Enter / Space). External scripts should listen to this event to handle button actions.

button.addEventListener('button-action',() =>{
 console.log('Button clicked');
});

Attributes

aria-label - static (required). Must be set manually for icon buttons.

disabled - reactive (optional). Controls the disabled state of the component.

busy - reactive (optional). Controls the busy state of the component.

shape - static (optional). Available options: pill, circle, square.

size - static (optional). Available options: x-small, small, medium, large, x-large. Default: medium
