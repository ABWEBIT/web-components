Button

Buttons are used to perform actions or submit forms.

Info

The button can contain text, icons, or both - in any order.

Attributes

data-radius - Available options: full, none.

data-size - Available options: small, medium (default), large.

data-color - Available options: blue, green, yellow, red.

aria-label - Must be set manually for icon buttons.
