Radio

Allows the user to select only one option from a set of choices.

Attributes

config - static (optional). Additional attributes: id, name, value, class.

checked - reactive (optional). Default: false.

required - reactive (optional). Default: false.

disabled - reactive (optional). Default: false.

size - static (optional). Controls the overall size of the element. Available options: sm, md (default), lg.

Default


<ui-radio></ui-radio>

Config

Each property in the JSON config must be enclosed in double quotes, both property names and their values, except for the boolean values true and false.

<ui-radio
  config='{
    "id":"unique-id",
    "class":"utility-class",
    "name":"product-id",
    "value":"on"
  }'>
</ui-radio>

Label

There are two ways to add a label to a component: either by wrapping the component inside a <label>, or by linking a separate <label> to the component using the for attribute with the component's id.

<label>
  Label<ui-radio></ui-radio>
</label>

<label for="input">Label</label>
<ui-radio config='{"id":"input"}'></ui-radio>
