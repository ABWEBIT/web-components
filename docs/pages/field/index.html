Field

The Field component groups a div with a form control.

Info

Allows only one label and one form control inside.

Automatic association between the label and the form control by adding a unique UUID.

Attributes

vertical - static (optional). Sets the orientation to vertical.

required - static (optional). Adds the required attribute to all child elements.

Default


<ui-field>
  <label>Label</label>
  <ui-checkbox></ui-checkbox>
</ui-field>

<ui-field>
  <label>Label</label>
  <ui-radio></ui-radio>
</ui-field>

  <label>Label</label>

</ui-field>
