Breadcrumb

Breadcrumbs show the current page's location within a navigational hierarchy and allow users to navigate to parent pages.

Attributes

  • theme - static (optional). Default: default.
    Specifies the component's color scheme (extendable via CSS).
Text only
HTML 

<ui-breadcrumb id="breadcrumb"></ui-breadcrumb>

breadcrumb.data = [
 {label:'Home',href:'/'},
 {label:'Settings',href:'/'},
 {label:'Notifications',href:'/'},
 {label:'Article',href:'/'}
];
Text with icons
HTML 

<ui-breadcrumb id="breadcrumb"></ui-breadcrumb>

breadcrumb.data = [
 {icon:'home',label:'Home',href:'/'},
 {icon:'settings',label:'Settings',href:'/'},
 {icon:'notifications',label:'Notifications',href:'/'},
 {icon:'article',label:'Article',href:'/'}
];
