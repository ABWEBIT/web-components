Breadcrumb

Breadcrumbs show the current page's location within a navigational hierarchy and allow users to navigate to parent pages.

Attributes

  • text-align - static (optional). Default: start.
    Available options: start, center, end, justify.
    Controls text alignment.
Start
HTML
<ui-breadcrumb></ui-breadcrumb>

items = [{
 label:'Home',
 href:'/'
},{
 label:'Level 1',
 href:'/level_1'
},{
 label:'Level 2',
 href:'/level_1/level_2'
},{
 label:'Level 3',
 href:'/level_1/level_2/level_3'
}];
