Breadcrumb

Breadcrumbs show the current page's location within a navigational hierarchy and allow users to navigate to parent pages.

Object properties

  • icon - optional.
    An icon that can be displayed before the text in a breadcrumb item.
  • title - required.
    The text displayed in the breadcrumb item.
  • href - required.
    The URL of the breadcrumb item.
Text only
HTML 

<ui-breadcrumb id="breadcrumb"></ui-breadcrumb>

breadcrumb.data = [
 {title:'Home',href:'/'},
 {title:'Settings',href:'/'},
 {title:'Notifications',href:'/'},
 {title:'Article',href:'/'}
];
Text with icons
HTML 

<ui-breadcrumb id="breadcrumb"></ui-breadcrumb>

breadcrumb.data = [
 {icon:'home',title:'Home',href:'/'},
 {icon:'settings',title:'Settings',href:'/'},
 {icon:'notifications',title:'Notifications',href:'/'},
 {icon:'article',title:'Article',href:'/'}
];
