Select

A Select is a component that lets the user choose a single option from a list of multiple choices.

Attributes

  • aria-label - static (required).
    If used outside of the field component, the aria-label must be set manually.
  • placeholder - static (optional). Default: "-".
    Static text to display when no item is selected.
  • icon - static (init-only, optional). Default: arrow-down-small.
  • size - static (optional). Default: default.
    Specifies the component's sizes (extendable via CSS).
  • color - static (optional). Default: default.
    Specifies the component's color scheme (extendable via CSS).
  • shape - static (optional). Default: rounded.
    Available options: rounded, pill, square.
    Sets the corner style.
