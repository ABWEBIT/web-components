Label

A label provides accessible, descriptive text for a user interface control.

Usage

  • The component is used in conjunction with a form control and a field.

Features

  • By default, the cursor is set to pointer.

Attributes

  • passive - static, optional.
    Sets the cursor to default, giving the impression that the element is not interactive.
  • required - static, optional.
    Indicates whether the associated form control is required.
  • symbol - static, optional. Default: *.
    Overrides the default required symbol. Only a single character is allowed; if the string is longer than one character, only the first character will be used. The symbol is added only if the required attribute is present.
