Separator

A component for visually separating content with a line.

Usage

  • Any content added inside the component should be wrapped in a <span> to ensure proper rendering.
  • When using non-text content (e.g., icons or custom elements), additional CSS may be required to ensure correct styling.

Features

  • By default, a horizontal separator has a width of 100%.
  • By default, a vertical separator has a height of 100%.

Attributes

  • orientation - static (optional). Default: horizontal.
    Available options: horizontal, vertical.
    Defines the separator direction.
  • theme - static (optional). Default: default.
    Specifies the component's color scheme (extendable via CSS).
Horizontal separator
HTML
<ui-separator></ui-separator>
Vertical separator
HTML
<ui-separator orientation="vertical"></ui-separator>
