Accordion

Accordion toggles collapsible content sections.

Attributes

  • size - static (optional). Default: default.
    Specifies the component's sizes (extendable via CSS).
  • theme - static (optional). Default: default.
    Specifies the component's color scheme (extendable via CSS).
  • shape - static (optional). Default: rounded.
    Available options: rounded, square.
    Sets the corner style.

Object properties

  • title - required.
    The text displayed in the accordion header.
  • content - required.
    The HTML or text content shown inside the expanded panel.
  • disabled - optional.
    If true, the item is non-interactive and cannot be toggled.
  • expanded - optional.
    If true, the panel is expanded by default on initial render.
Minimal
HTML 

<ui-accordion id="accordion"></ui-accordion>

accordion.data = [
 {title:'What is Lorem?',content:'Lorem ipsum dolor sit amet...',expanded:true},
 {title:'Why use it?',content:'Lorem ipsum dolor sit amet...',disabled:true},
 {title:'Where to use?',content:'Lorem ipsum dolor sit amet...'}
];
