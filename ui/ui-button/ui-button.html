Button

Buttons are used to perform actions or submit forms.

Usage

  • The component functions as a wrapper. It can contain text, icons, or both - in any order.
  • This component is designed to work with the ui-icon component for rendering icons, but it can easily be adapted to support raw SVGs or other elements if needed.
  • Since the default icons have 2px padding inside the SVG, --ui-button-icon-margin-inline-trim: -2px is applied in size="default" to visually balance the button's side spacing. This affects only the width - not the height. If your icons extend to the full SVG width, you should remove this adjustment.

Attributes

  • aria-label - static (init-only, required).
    Must be set manually for icon-only buttons.
  • type - static (init-only, optional).
    Controls the disabled state of the component.
  • disabled - reactive (optional).
    Controls the disabled state of the component.
  • loading - reactive (optional).
    Controls the loading state of the component.
  • size - static (optional). Default: default.
    Specifies the component's size (extendable via CSS).
    size="icon-only" - for a button with symmetrical sides that contains only an icon.
  • theme - static (optional). Default: default.
    Specifies the component's color scheme (extendable via CSS).
  • shape - static (optional). Default: rounded.
    Available options: rounded, pill, square, circle.
    Sets the corner style.
