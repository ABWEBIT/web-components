Button

Buttons are used to perform actions or submit forms.

Usage

  • The component can contain text, icons, or both - in any order.
  • This component is designed to work with the icon component or an SVG element for rendering icons.
  • This component dispatches a custom event "button-action" whenever it is activated via click or keyboard (Enter / Space). External scripts should listen to this event to handle button actions.

    Example:
    button.addEventListener('button-action',() =>{
     console.log('Button clicked');
    });

Attributes

  • aria-label - static (required).
    Must be set manually for icon buttons.
  • disabled - reactive (optional).
    Controls the disabled state of the component.
  • loading - reactive (optional).
    Controls the loading state of the component.
  • icon-only - static (optional).
    Indicates that the button contains only an icon, without text.
    The button will be displayed as a square (with symmetrical sides).
  • shape - static (optional). Default: rounded.
    Available options: rounded, pill, square, circle.
    Sets the corner style.
