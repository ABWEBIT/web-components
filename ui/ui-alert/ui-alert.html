Alert

Displays a message that draws the user's attention.

Features

Attributes

  • size - static (optional). Default: default.
    Specifies the component's size (extendable via CSS).
  • theme - static (optional). Default: default.
    Specifies the component's color scheme (extendable via CSS).
  • shape - static (optional). Default: rounded.
    Available options: rounded, square.
    Sets the corner style.
Alert with Label only
HTML
<ui-alert></ui-alert>

element.setData({
 label:'Only label and is required'
});
Alert with Label and Description
HTML
<ui-alert></ui-alert>

element.setData({
 label:'Alert',
 content:'Description goes here'
});
Alert with Label, Description and Icon
HTML
<ui-alert></ui-alert>

element.setData({
 icon:'info',
 label:'Information notice',
 content:'Please be aware that system maintenance is scheduled for tonight at 19:00.'
});
