Alert

Displays a message that draws the user's attention.

Usage

  • All attributes are optional, but the component must have either a label or content to display a meaningful message.

Attributes

  • label - static (init-only, optional).
    The main heading or title of the alert.
  • content - static (init-only, optional).
    Additional message below the label.
  • icon - static (init-only, optional).
    Name of icon to display.
  • closable - static (optional).
    If present, displays a close button.
  • size - static (optional). Default: default.
    Specifies the component's size (extendable via CSS).
  • theme - static (optional). Default: default.
    Specifies the component's color scheme (extendable via CSS).
  • shape - static (optional). Default: rounded.
    Available options: rounded, square.
    Sets the corner style.
Alert with Label only
HTML
<ui-alert
 label="Alert with Label only">
</ui-alert>
Alert with Content only
HTML
<ui-alert
 content="Alert with Content only">
</ui-alert>
Alert with Label and Content
HTML
<ui-alert
 label="Alert"
 content="Alert with Label and Content">
</ui-alert>
Alert with Label, Content and Icon
HTML
<ui-alert
 label="Update Complete"
 content="Your settings have been successfully saved."
 icon="info">
</ui-alert>
Alert with Label, Content, Icon and Close Button
HTML
<ui-alert
 label="System Alert"
 content="The system will be unavailable from 19:00."
 icon="warning"
 closable>
</ui-alert>
