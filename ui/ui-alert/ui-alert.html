Alert

Displays a message that draws the user's attention.

Usage

  • The component requires either the title or description to display a meaningful message.
  • HTML markup is supported in the title and description fields for richer formatting.
  • The component uses the ui-icon component to render icons.

Attributes

  • icon - static (init-only, optional).
    Name of icon to display.
  • no-close - static (optional, boolean).
    If present, the close button is not rendered.
  • size - static (optional). Default: default.
    Specifies the component's size (extendable via CSS).
  • theme - static (optional). Default: default.
    Specifies the component's color scheme (extendable via CSS).
  • shape - static (optional). Default: rounded.
    Available options: rounded, square.
    Sets the corner style.
