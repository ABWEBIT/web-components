Text

Standardizes and ensures consistent text styling.

Features

  • The text block width is 100%.

Attributes

  • text-align - static (optional). Default: start.
    Available options: start, center, end, justify.
    Controls text alignment.
  • theme - static (optional). Default: default.
    Specifies the component's color scheme (extendable via CSS).
