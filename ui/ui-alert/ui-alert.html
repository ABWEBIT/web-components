Alert

Displays a message that draws the user's attention.

Usage

  • The component must be composed of the following containers: ui-icon, ui-title, and ui-content. While their order is not strictly required, following the recommended sequence ensures better consistency and styling.
    Recommended order: ui-icon - ui-title - ui-content.
  • Either a title or a description is required to display a meaningful message.
  • This component is designed to work with the ui-icon component for rendering icons.

Attributes

  • closable - static (optional, boolean).
    Renders a close button when present.
  • size - static (optional). Default: default.
    Specifies the component's size (extendable via CSS).
  • theme - static (optional). Default: default.
    Specifies the component's color scheme (extendable via CSS).
  • shape - static (optional). Default: rounded.
    Available options: rounded, square.
    Sets the corner style.
