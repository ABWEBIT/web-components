Alert

Displays a message that draws the user's attention.

Usage

  • Neither element is mandatory; however, either a title or a description is required to display a meaningful message.
  • This component is designed to work with the ui-icon component for rendering icons.

Attributes

  • non-closable - static (optional, boolean).
    Hides the close button when the attribute is present.
  • shape - static (optional). Default: rounded.
    Available options: rounded, square.
    Sets the corner style.
Alert with description, without a close button
HTML
<ui-alert id="ex00" non-closable></ui-alert>

ex01.data = {
 description: 'Your settings have been <strong>successfully</strong> saved.'
};
Alert with title and description
HTML
<ui-alert id="ex01"></ui-alert>

ex01.data = {
 title: 'Update Complete',
 description: 'Your settings have been <strong>successfully</strong> saved.'
};
Alert with title, description, close button, and icon
HTML
<ui-alert id="ex02" icon="info"></ui-alert>

ex02.data = {
 title: 'Update Complete',
 description: 'Your settings have been <strong>successfully</strong> saved.'
};
