Field

The Field component groups a label with a form control.

Usage

  • Only two components are allowed inside: a ui-label and a form control.

Features

  • Automatic association between the label and the form control by adding a unique UUID to the label's id attribute and assigning this UUID to the form control's aria-labelledby attribute. You can override the unique UUID by adding a custom id attribute to the ui-label component.

Attributes

  • label-position - static (optional). Default: start.
    Defines the position of the ui-label relative to the form control. By default, the label is placed before the form control, so start doesn't need to be explicitly set. Use end to place the label after the control.
  • orientation - static (optional). Default: horizontal.
    Defines the layout direction of the label and the form control. horizontal | vertical
  • required - static (optional).
    Adds the required attribute to all child elements.
