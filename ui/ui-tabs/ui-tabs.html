Tabs

Tabs organize content into separate panels that users can switch between.

Attributes

  • shape - static (optional). Default: rounded.
    Available options: rounded, square.
    Sets the corner style.

Object properties

  • title - required.
    The text displayed in the tab.
  • content - required.
    The HTML or text content shown inside the panel.
Rounded shape with a disabled tab
HTML 

<ui-tabs id="tabs"></ui-tabs>

tabs.data = [
 {label:'What is Lorem?',content:'Lorem ipsum...'},
 {label:'Why use it?',content:'Lorem ipsum...',disabled:true},
 {label:'Where to use?',content:'Lorem ipsum...'}
];
