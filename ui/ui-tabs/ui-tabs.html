Tabs

Tabs organize content into separate panels that users can switch between.

Attributes

  • size - static (optional). Default: default.
    Specifies the component's size (extendable via CSS).
  • theme - static (optional). Default: default.
    Specifies the component's color scheme (extendable via CSS).
  • shape - static (optional). Default: rounded.
    Available options: rounded, square.
    Sets the corner style.
Rounded shape with a disabled tab
HTML 

<ui-tabs id="tabs"></ui-tabs>

tabs.data = [
 {label:'What is Lorem?',content:'Lorem ipsum dolor sit amet...'},
 {label:'Why use it?',content:'Lorem ipsum dolor sit amet...',disabled:true},
 {label:'Where to use?',content:'Lorem ipsum dolor sit amet...'}
];
