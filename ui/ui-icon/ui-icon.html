Icon

A component for displaying an icon.

Features

  • The SVG uses currentColor for both fill and stroke.

Usage

  • For filled icons, ensure <path> has stroke="none" to prevent unintended strokes from global CSS.
  • Line icons may require additional CSS for stroke settings.

Attributes

  • icon - reactive (required).
    Specifies the name of the icon to display.
