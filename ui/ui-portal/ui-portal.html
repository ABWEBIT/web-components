Portal

A root-level DOM node for modals, overlays, and dialogs.

Usage

  • Place just before the closing </body> tag.

Features

  • Ensures the component is outside the main document flow, avoids z-index conflicts, and guarantees that modals, overlays, and dialogs render above all other content.
Placement
HTML
<body>
 <!-- Preceding code -->
 <ui-portal></ui-portal>
</body>
