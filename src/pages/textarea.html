Textarea

Textarea allows users to enter and edit multiline text.

Attributes

config - static (optional). Additional attributes.

required - reactive (optional). Defines whether the component is required.

disabled - reactive (optional). Controls the disabled state of the component.

readonly - reactive (optional). Controls the readonly state of the component.

value - reactive (optional). Sets or binds the content of the <textarea>

radius - static (optional). Default: rounded. Available options: none. Sets the corner style.

Default


<ui-textarea></ui-textarea>

Config

Each property in the JSON config must be enclosed in double quotes, both property names and their values, except for the boolean values true and false.

<ui-input
  config='{
    "id":"unique-id",
    "class":"utility-class",
    "name":"product-id",
    "placeholder":"Enter text",
    "rows":"3",
    "cols":"10",
    "maxlength":"50",
    "minlength":"3",
    "wrap":"soft",
    "autocomplete":"off",
    "inputmode":"soft",
    "spellcheck":"false",
    "autofocus":true
  }'>
</ui-input>
