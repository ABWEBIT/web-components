Accordion

Accordion organizes content into collapsible sections.

Dependencies

  • ui-icon

Info

The structure must be strictly followed.

By default, the accordion allows multiple items to be expanded, but it can be set to single-item mode.

The heading can be any HTML heading element from h2 to h6; h3 is recommended.

The accordion does not support dynamically adding or removing items.

Attributes

Accordion:

single - static (optional). Allows only one expanded item at a time.

Item:

disabled - reactive (optional). Determines if the item is disabled.

expanded - reactive (optional). The associated panel is expanded.
