Button

Buttons are used to perform actions or submit forms.

Attributes

radius - static (optional). Controls how rounded the corners are. Available options: full, none.

size - static (optional). Controls the overall size of the element. Available options: sm, md (default), lg.

color - static (optional). Controls the color style of the element. Available options: gray, red, green, blue, amber.

Config

Each property inside the config JSON string must be enclosed in double quotes. This applies both to the property names and their values.
Button 

<ui-button
  config='{
    "id":"unique-id",
    "class":"utility-class",
    "type":"submit"
  }'>
</ui-button>
