Select

A Select is a component that lets the user choose a single option from a list of multiple choices.

Dependencies

  • ui-icon
  • ui-button

Attributes

aria-label - static (required). If used outside of the ui-field component, the aria-label must be set manually.

placeholder - static (optional). Default: "-". Static text to display when no item is selected.

icon - static (init-only, optional). Default: arrow-down-small.

shape - static (optional). Default: rounded. Available options: rounded, pill, square. Sets the corner style.
